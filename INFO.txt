Instalar variables de entorno
npm i dotenv

Instalar el cors (Para que me acepte peticiones de diferentes dominios)
npm i cors

Paquete de Node para express de validadores predefinidos
npm i express-validator

Encriptar contraseñas
npm i bcryptjs

General JWT (JSON WEB TOKEN)
npm i jsonwebtoken

Subir archivos con express
npm i express-fileupload

Para crear identificadores únicos
npm install uuid
